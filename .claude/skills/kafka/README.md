# Apache Kafka Skill

This skill provides comprehensive guidance for building event-driven architectures using Apache Kafka from hello world examples to professional production systems.

## Overview

The Kafka skill contains:

- **SKILL.md**: Main skill file with comprehensive documentation
- **scripts/**: Utility scripts for common Kafka operations
- **references/**: Detailed reference materials
- **assets/**: Example code and templates

## Features

- Complete producer and consumer examples in Java and Python
- Topic management with Admin API
- Security configuration (SSL, SASL)
- Monitoring and observability guidance
- Production best practices
- Stream processing with Kafka Streams
- Kafka Connect configuration
- Troubleshooting guidance

## Usage

When working with Kafka-related tasks, this skill will provide:

- Code examples and templates
- Best practices for different scenarios
- Configuration guidance
- Troubleshooting assistance
- Security recommendations
- Performance optimization tips
