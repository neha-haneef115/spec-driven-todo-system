=== Verify Deployment ===

Check all pods are running:
  kubectl get pods -n {{ .Release.Namespace }}

Check services:
  kubectl get svc -n {{ .Release.Namespace }}

{{ if .Values.dapr.enabled }}
Check DAPR sidecars are running:
  kubectl get pods -n {{ .Release.Namespace }} -o=jsonpath='{range .items[*]}{.metadata.name}{"\t"}{range .spec.containers[*]}{.name}{"\n"}{end}{end}' | grep -E '(daprd|dapr)'

Check DAPR status:
  dapr status -k

Interact with DAPR:
  # Invoke services via DAPR
  dapr invoke --app-id {{ include "taskflow.fullname" . }}-backend --method [method]

  # Publish events
  curl -X POST http://localhost:3500/v1.0/publish/[pubsub-name]/[topic] -H "Content-Type: application/json" -d '[payload]'
{{ end }}

View backend logs:
  kubectl logs -f deployment/{{ include "taskflow.fullname" . }}-backend -n {{ .Release.Namespace }}

View frontend logs:
  kubectl logs -f deployment/{{ include "taskflow.fullname" . }}-frontend -n {{ .Release.Namespace }}

View MCP server logs:
  kubectl logs -f deployment/{{ include "taskflow.fullname" . }}-mcp-server -n {{ .Release.Namespace }}

=== Important Notes ===

Make sure you have set the environment variables in secrets.yaml

{{ end }}

=== Uninstall ===

To uninstall the chart:
  helm uninstall {{ .Release.Name }} -n {{ .Release.Namespace }}

To also delete the namespace:
  kubectl delete namespace {{ .Release.Namespace }}
